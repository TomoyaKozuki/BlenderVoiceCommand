# BlenderVoiceCommand
こちらはBlenderを音声認識によって操作する拡張機能を実装しました。

# Requirements
- blender 4.1
- blender python 3.11.7
## Python dependency 
- pywhispercpp
- faster whisper
- numpy
- sounddevice
- janome

# Usage
⚠blenderを起動する際には、「管理者権限で実行」を押して下さい
